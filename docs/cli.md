# CLI Tool

The CLI tool allows you to interact with the AI directly from the command line.

## Usage

```bash
npm run cli "Your question here"
```

## Examples

```bash
npm run cli "What is the capital of France?"
npm run cli "Explain quantum computing"
```

## Requirements

- Node.js 20+
- GEMINI_API_KEY in .env

## Implementation

The CLI uses the same AI backend as the web app, providing consistent responses.
